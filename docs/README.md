## 🔧 配置文件说明

所有功能都在 `hugo.toml` 中配置，主要配置项：

```toml
[params]
  # 视频支持
  [params.video]
    enable = true
  
  # 轮播图
  [params.carousel]
    enable = true
    # ...更多配置
  
  # 打赏按钮
  [params.reward]
    enable = true
    # ...更多配置
  
  # 侧栏配置
  [params.aside]
    right = ["series-posts", ...]
    
    # 系列文章
    [params.aside.seriesPosts]
      enable = true
      # ...更多配置
```

**详细配置请查看各功能的使用指南。**

## ❓ 常见问题

### 功能不生效？

1. **检查 `enable` 配置**
   - 所有功能都有 `enable` 开关
   - 确保设置为 `true`

2. **重新构建**
   ```bash
   hugo --cleanDestinationDir
   hugo server
   ```

3. **清除缓存**
   - 浏览器强制刷新（Ctrl+F5）
   - 清除浏览器缓存

### 样式错乱？

1. **检查 CSS 加载**
   - 打开浏览器开发者工具
   - 查看 Network 标签
   - 确认 CSS 文件加载成功

2. **检查 Hugo 版本**
   - 建议使用 Hugo v0.100+
   - 某些功能需要较新版本

### 多个功能冲突？

1. **逐个启用测试**
   - 一次启用一个功能
   - 确认无问题后再启用下一个

2. **查看浏览器控制台**
   - 检查是否有 JavaScript 错误
   - 查看错误信息
