**6. 文章内容增强功能 ✨**

新增以下子功能的详细说明：

##### 音乐播放器 🎵

```markdown
<!-- 网易云音乐 -->
{{< music "netease:歌曲ID" >}}

<!-- 歌单 -->
{{< music "netease-playlist:歌单ID" >}}

<!-- 本地音乐 -->
{{< music "local:audio/song.mp3" >}}

<!-- 混合播放 -->
{{< music 
  "netease:591321" 
  "local:audio/demo.mp3" 
  "url:https://example.com/song.mp3" 
>}}
```

##### 彩色文字 🎨

```markdown
<!-- 预设颜色 -->
{{< color "red" >}}红色文字{{< /color >}}

<!-- 自定义颜色 -->
{{< color "#FF5733" >}}自定义颜色{{< /color >}}

<!-- 主题色 -->
{{< color "primary" >}}主题色{{< /color >}}
```

##### 评论可见内容 🔒

```markdown
{{< reply-visible id="unique-id" >}}
评论后才能看到的内容
{{< /reply-visible >}}
```

##### 内容折叠 📦

```markdown
{{< collapse "点击展开" >}}
折叠的内容
{{< /collapse >}}

<!-- 默认展开 -->
{{< collapse "标题" "open" >}}
默认展开的内容
{{< /collapse >}}
```

##### 时间线 📅

```markdown
{{< timeline >}}

{{< timeline-item "2025-10-25" "标题" "success" "star" >}}
时间线内容
{{< /timeline-item >}}

{{< /timeline >}}
```

**可用类型**：default, success, warning, danger, info  
**可用图标**：default, check, star, warning, info, plus

##### 选项卡 📑

```markdown
{{< tabs >}}

{{< tab "选项卡1" >}}
第一个选项卡的内容
{{< /tab >}}

{{< tab "选项卡2" >}}
第二个选项卡的内容
{{< /tab >}}

{{< /tabs >}}
```

**7. 页脚配置 ⏱️**

新增页脚配置说明：

```toml
[params.footer]
  # 自定义内容
  custom = """
    <div class="footer-links">
      <a href="/about">关于</a>
      <a href="/sitemap.xml">网站地图</a>
    </div>
  """
  
  # 运行时间显示
  [params.footer.runningTime]
    enable = true
    startDate = "2024-01-01"  # 建站日期
    prefix = "本站已运行"       # 前缀文字
```

**显示效果**：
- 版权信息（自动显示年份范围）
- 运行时间（动态更新，精确到秒）
- 自定义链接
- 备案信息（可选）
